=LabPixies Widget= This widget allows you to add the [http://www.labpixies.com/ LabPixies] widgets to your wiki page. ==Usage== {{#widget:Labpixies |url=widget-url |allowtransparency=true OR false |width=width |height=height }} ===Parameters=== {|class=datatable ! Parameter !! Notes !! Default Value |- | url || The url ''fragment'' of the widget || |- |allowtransparency || allowTransparency attribute || true |- | width || iframe's width || 280 |- | height || iframe's height || 202 |} ===URL Fragment=== ===More information=== Visit [http://www.labpixies.com] for more information. Each widget page is presented with an ''< embed code >'' button where the ''url-fragment'' can be found i.e. look at the iframe's src element. == Example == {{#widget:Labpixies |url=/babylon/babylon.html }} {{#widget:Labpixies |url=/babylon/babylon.html }} ==Code== Provided as reference only.
