=Addthis Widget= This widget allows you to add the [http://www.addthis.com/ AddThis] social-bookmarking button to your wiki page. ==Credit== This widget was inspired from work done by [http://www.mediawiki.org/wiki/User:Yaron_Koren Yaron Koren]. ==Usage== {{#widget:Addthis |page_name={{PAGENAME}} |page_url={{fullurl:{{PAGENAME}}}} |account_id=my-account-id |logo_url= |logo_background=FFFFFF |logo_color=FFFFFF |brand=My Wiki |options=favorites, email, digg, delicious, more |offset_top=0 |offset_left=0 }} ===Parameters=== {|class=datatable ! Parameter !! Notes !! Default Value |- | page_name || The page name which will appear in the bookmark || {{PAGENAME}} |- | page_url || The page's URL to bookmark || {{fullurl:{{PAGENAME}}}} |- | account_id || The account-id to which to bookmarks will be attributed to || none |- | logo_url || The site logo [optional] || none |- | logo_bkg || The logo's background color || FFFFFF |- | logo_color || The logo's color || FFFFFF |- | brand || The ''brand'' name that should appear in the badge || {{SITENAME}} |- | options || The bookmarking sites || favorites, email, digg, delicious |- | offset_top || offset at the top || 0 |- | offset_left || offset from the left || 0 |} ===Usage Notes=== You are encouraged to use a template (e.g. [[Template:Addthis]] to use this widget. ===More information=== You can read more about these options on [http://www.addthis.com/customization.html the AddThis documentation page]. == Example == {{#widget:{{PAGENAME}} |page_name={{PAGENAME}} |page_url={{fullurl:{{PAGENAME}}}} |account_id=jldupont |logo_url=/favicon.ico |logo_background=FFFFFF |logo_color=FFFFFF |brand={{SITENAME}} |options=favorites, email, digg, delicious, more |offset_top=0 |offset_left=0 }} ==Code== Provided as reference only.
